# Responsive Web Projects

A collection of frontend web development projects built during a 
virtual internship. All projects follow a consistent dark navy blue 
theme and are built using HTML5, CSS3, and vanilla JavaScript.

## Projects

### Week 1 - Responsive Landing Page
A fully responsive landing page for CyberShield, a fictional 
cybersecurity product. Includes sticky navigation, hero section, 
features grid, contact form, and footer.

### Week 2 - Interactive FAQ Accordion
An interactive FAQ accordion component with smooth animations. 
Only one answer opens at a time. Built with vanilla JavaScript 
DOM manipulation.

### Week 3 - Accessibility Enhancements
Accessibility improvements applied to the Week 1 landing page 
following WCAG and ARIA guidelines. Includes skip navigation, 
focus styles, and keyboard support.

### Week 4 - Performance Optimization
Performance optimizations applied including minified CSS and JS, 
async script loading, and meta description tag.

### Week 5 - CyberShield Dashboard
A mini web application dashboard that loads threat data from a 
static JSON file and displays it in an interactive table with 
risk level filtering.
