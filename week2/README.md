\# CyberShield FAQ Accordion



\## What is this?

An interactive FAQ accordion component built with HTML, CSS, and vanilla JavaScript.



\## How to use

1\. Open index.html in any browser

2\. Click any question to expand the answer

3\. Click again to collapse it

4\. Only one answer stays open at a time



\## Files

\- index.html — structure

\- style.css — styling

\- script.js — interactivity



\## Features

\- Smooth open/close animation

\- Only one item open at a time

\- Accessible with aria-expanded attribute

\- Responsive on all screen sizes
